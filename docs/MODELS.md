# Модели

## UserProfile
Профиль пользователя, расширяющий стандартную модель Django User

| Поле | Тип | Описание |
|------|-----|----------|
| user | OneToOneField(User) | Связь со стандартной моделью пользователя |
| bio | TextField | Краткая информация о пользователе (до 500 символов) |
| image | ImageField | Фото профиля (макс. 2MB, форматы: jpg, jpeg, png, gif) |

---

## Course
Учебный курс, созданный преподавателем

| Поле | Тип | Описание |
|------|-----|----------|
| title | CharField(200) | Название курса (от 5 до 200 символов) |
| description | TextField | Подробное описание курса (минимум 50 символов) |
| created_at | DateTimeField | Дата создания (автоматически) |
| price | DecimalField | Цена в рублях (от 0 до 1 000 000) |
| published | BooleanField | Статус публикации (опубликован/скрыт) |
| author | ForeignKey(UserProfile) | Автор курса (преподаватель) |

**Ограничения:**
- Один автор не может создать два курса с одинаковым названием
- Неопубликованные курсы видны только автору
- Нельзя записаться на неопубликованный курс

---

## Lesson
Урок внутри курса

| Поле | Тип | Описание |
|------|-----|----------|
| course | ForeignKey(Course) | Курс, к которому относится урок |
| name | CharField(200) | Название урока (от 3 до 200 символов) |
| description | TextField | Краткое описание урока |
| video_url | URLField | Ссылка на видеоурок |
| content | TextField | Основной контент урока (текст, задания) |
| order | PositiveIntegerField | Порядковый номер урока в курсе |

**Ограничения:**
- Уроки автоматически сортируются по полю order
- Порядковый номер уникален в рамках одного курса
- Контент урока обязателен для заполнения

---

## Enrollment
Запись студента на курс

| Поле | Тип | Описание |
|------|-----|----------|
| user | ForeignKey(User) | Студент, записавшийся на курс |
| course | ForeignKey(Course) | Курс, на который записан студент |
| date_enrolled | DateTimeField | Дата записи (автоматически) |
| status | CharField | Статус записи: `active` (активна), `cancelled` (отменена), `completed` (завершена) |

**Ограничения:**
- Один студент может записаться на курс только один раз
- Автор курса не может записаться на свой собственный курс
- Нельзя записаться на неопубликованный курс

---

## LessonProgress
Прогресс прохождения уроков студентом

| Поле | Тип | Описание |
|------|-----|----------|
| user | ForeignKey(User) | Студент, проходящий урок |
| lesson | ForeignKey(Lesson) | Урок, который проходится |
| is_completed | BooleanField | Статус прохождения: `True` (пройден), `False` (в процессе) |
| completed_at | DateTimeField | Дата и время завершения урока (автоматически) |

**Ограничения:**
- Один студент может иметь только один прогресс на конкретный урок
- Связь `(user, lesson)` уникальна

**Назначение:**
- Отслеживание прогресса студента по каждому уроку
- Используется для расчёта общего прогресса по курсу (через сервис `course_progress`)

---

## Связи между моделями

| Связь | Тип | Описание |
|-------|-----|----------|
| User → UserProfile | One-to-One | Один пользователь — один профиль |
| UserProfile → Course | One-to-Many | Один автор — много курсов |
| Course → Lesson | One-to-Many | Один курс — много уроков |
| User → Enrollment | One-to-Many | Один студент — много записей на курсы |
| Course → Enrollment | One-to-Many | Один курс — много записей студентов |
| User → LessonProgress | One-to-Many | Один студент — прогресс по многим урокам |
| Lesson → LessonProgress | One-to-Many | Один урок — прогресс многих студентов |
